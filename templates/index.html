Corona Tracker REST API

by Swimmers

Play Store and Apple store is not allowing any one to publish corona related apps to ensure no fake information to flow regarding that.

So You can try this app from here. All the information are valid and are from worldometer.info but don't rely on it so far

They will be valid until worldometer is valid

GET /all

You will get all the informations about COVID-data around the world.

It gives list of countries

GET /country/[country-name]

You can get information of a particular country

Please see country-name list in Country Codes page

Note: If new country records are not available in country code list

[/country/Total:] gives information of world

GET /updates/[country-code]

Get updated informations for particular countries

Please see country-code list in Country Codes page

Note: If country is not listed in code list. Try to add "[new]" at the end of country name

[/update/world] gives information of world

GET /updates

Search for all countries and get only updated informations

GET /countrycode/[country-code]

Get by country codes

Please see country-code list in Country Codes page

Note: If country is not listed in code list. Try to add "[new]" at the end of country name

[/countrycode/world] gives information of world
